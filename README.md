# Uncertainty-Aware QA System

A BERT-based Question Answering system with uncertainty-aware reliability scoring.

## Features
- BERT QA model
- Entropy-based uncertainty estimation
- Random Forest reliability classifier
- Reliability score prediction

## Tech Stack
- Python
- PyTorch
- HuggingFace Transformers
- Scikit-learn
- Pandas

## Pipeline
Question + Context → BERT → Feature Extraction → Random Forest → Reliability Score
